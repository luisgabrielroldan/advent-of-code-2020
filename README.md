# Advent of Code 2020

https://adventofcode.com/2020


### Running solutions

The puzzle inputs are on the [input](input/) folder.

To get the solutions for a day run:

```
cargo run [day]
```

Example:

```
cargo run 1

...


*=============================* 
|     Advent of Code 2020     |
*=============================*

```
